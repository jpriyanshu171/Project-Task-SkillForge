# SkillForge Frontend

This frontend is a React 18 + Vite + Tailwind application for the SkillForge platform.

## Setup

1. Copy `.env.example` to `.env.local`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`

## Features

- Authentication pages
- Dashboard analytics
- Skills and recommendations pages
- JWT integration with backend API
- React Query caching and performance-aware design
